Spring Cloud Example

使用Spring Cloud来搭建微服务框架，Spring Cloud提供了一套基本方案，用于简化微服务之间衔接时的开发。核心组件是Netflix提供的
一整套开发套件。具体名词有zipKin, Hystrix with Hystrix DashBoard and Turbine, Zuul Eureka, Spring Cloud Config and so on.
在实现开发前，需要对这些名词有一定认识，知道微服务每一个环节可以采用的组件框架。

Go To Github

spring-cloud-example, 按照上图架构实现了基础代码
项目中基本不包含业务内容，仅基本框架的配置等信息。项目分以下四点来介绍了spring cloud 搭建微服务时可使用的基本套件的基本示例。

  • Spring Cloud Gateway Eureka Example

  • Spring Cloud Hystrix With Hystrix DashBoard And Turbine

  • Spring Cloud Sleuth And ZipKin Traces

  • Spring Cloud Config
