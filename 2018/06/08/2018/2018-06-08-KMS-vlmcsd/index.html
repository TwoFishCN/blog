KMS

KMS 简介

Key Management Service（简称:KMS），这个功能是在Windows Vista之后的产品中的一种新型产品激活机制， 目的是为了Microsoft更好的遏制非法软件授权行为(盗版)。

KMS 系统激活命令

命令中展示的IP地址应该是你的KMS服务器的IP地址。

KMS Office激活命令

敲命令之前需要确定你的系统是64位还是32位。

32位的情况

64位的情况

接下来的命令是一样的

同样的, 命令中的ip地址是你的KMS服务地址。

vlmcsd

简介

KMS Emulator in C (currently runs on Linux including Android, FreeBSD, Solaris, Minix, Mac OS, iOS, Windows with or without Cygwin)

GitHub Home

下载

可以直接通过release去下载当前的发布版本, 点击release打开Github的Release页面; 或直接下载2017年的版本, 点击vlmcsd-1111-2017-06-17-Hotbird64下载软件包。
