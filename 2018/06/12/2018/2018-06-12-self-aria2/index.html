Aria2 多线程下载工具

Aria2

简介

Aria2是一个轻量级的多协议和多源命令行下载实用程序。 它支持HTTP/HTTPS、FTP、SFTP、BitTorrent和Malalink。Aria2 可以通过内置的JSON-RPC和XML-RPC接口来操作。

上面是Aria2官方描述翻译版本， 说简单的就是Aria2是一个下载工具, 可以像迅雷一样使用。 甚至是百度网盘的资源（使用需要借助一些其他工具此处不重点介绍）

安装Aria2并启动服务

通过https://github.com/aria2/aria2/releases 连接你可以找到所有的发行版本。一般的，可以在页面看到如下的列表。

aria2-1.34.0-android-arm-build1.zip
aria2-1.34.0-win-32bit-build1.zip
aria2-1.34.0-win-64bit-build1.zip
aria2-1.34.0.tar.bz2
aria2-1.34.0.tar.gz
aria2-1.34.0.tar.xz
Source code (zip)
Source code (tar.gz)

我假设你是window10系统，那么下载aria2-1.34.0-win-32bit-build1.zip吧。这个可以很适合你。 下载后解压你会得到一个文件夹，文件夹里面的文件如下列表

aria2c.exe
AUTHORS
ChangeLog
COPYING
LICENSE.OpenSSL
NEWS
README.html
README.mingw

接下来，下载下面提供的连接文件。放在上面解压的文件夹里。

https://raw.githubusercontent.com/HwMuTou/BaiduExporter/master/aria2c/HideRun.vbs
https://raw.githubusercontent.com/HwMuTou/BaiduExporter/master/aria2c/README.md
https://raw.githubusercontent.com/HwMuTou/BaiduExporter/master/aria2c/aria2.conf
https://raw.githubusercontent.com/HwMuTou/BaiduExporter/master/aria2c/start.bat

然后，双击start.bat，Aria2下载服务就启动了。

Aria2下载最并发连接数限制16

Aria2默认的版本下载连接数被限制为16, 这导致了加速似乎有限。如果不限制会不会好很多呢。答案是肯定的。

如何获得不限制连接数的，或者说连接数限制数量更多的呢? 您需要自己修改源码，编译一个自己的版本。

在https://github.com/aria2/aria2/releases下载Source code (zip)。修改src/OptionHandlerFactory.cc里16这个数字。

修改之后编译一个定制的aria2c.exe
