Postgre SQL 集成版

背景

在使用Postgresql软件时一般都是直接安装，但是如果你想直接将数据库软件集成到你的软件中，那么使用解压即可使用的软件版本是再好不过了。

使用说明

下载PostgreSQL Binaries, zip 版本。https://www.postgresql.org/download/

下载之后解压，解压之后大致结构如下：

bin
doc
include
lib
pgAdmin 4
share
StackBuilder
symbols

使用方式和Mysql的集成版本类似，通过mysql命令初始一个Data目录，然后基于此Data目录来启动服务。

初始话数据库

start 启动数据库

stop  停止数据库

-D    数据库目录位置

-l    数据库软件日志记录位置

pgAdmin 4

pgAdmin会启动一个Web服务, Web服务提供类似 Navicat 软件的功能, 通过Web界面操作连接到数据库进行相关操作。软件包位于根目录位置 pgAdmin 目录下。

bin
docs
venv
web

在bin目录下pgAdmin4程序启动服务。之后通过浏览器访问pgAdmin的服务页即可。
