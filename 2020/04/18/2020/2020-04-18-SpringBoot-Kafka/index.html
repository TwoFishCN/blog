Spring Boot 特性 - Apache Kafka Support

Apache Kafka

Apache Kafka 是一个常用的分布式消息中间件，因其优异的性能被广泛采用。

Spring Boot 针对 Kafka 的支持

Spring Boot 有专门针对 Kafka 做针对性的支持。仅需要在项目中引入 spring-kafka 依赖就可以启用Spring Boot对 Kafka的支持。这里说的支持，主要是是 auto-configuration 。

发送消息到Kafka

当我们引入 spring-kafka 默认会在Spring中自动配置提供一个 kafkaTemplate，可以用来发送消息。

TIPS 配置项如果定义了spring.kafka.producer.transaction-id-prefix那么KafkaTransactionManager 会被自动 配置。另外，如果存在 RecordMessageConverter Bean 的定义，会被自动配置到 KafkaTemplate 中。

处理消息从Kafka

当Kafka基础环境提供出来之后，在Spring Boot中，可以在 Bean 的方法上注解 @KafkaListener 来创建监听点，用来处理消息数据。 如果环境中没有显示的提供 KafkaListenerContainerFactory, Spring Boot会自动提供一个基于配置spring.kafka.listener.* 默认的 KafkaListenerContainerFactory。

当环境中提供 KafkaTransactionManager Bean, 系统会自动把它关联到容器工厂 KafkaListenerContainerFactory. 类似的，如果提供了 ErrorHandler, AfterRollbackProcessor 和 ConsumerAwareRebalanceListener Bean的实例， 环境也会自动的默认的工厂实例。

更加Listener（监听器）的类型不同（批量，单个），会自动关联 RecordMessageConverter 或者 BatchMessageConverter。 如果把 RecordMessageConverter 提供给批处理监听器，它会自动被包装成 BatchMessageConverter。

TIPS 自定义 ChainedKafkaTransactionManager 必须标注 @Primary 注解， 因为系统通常会应用自动配置产生的 KafkaTransactionManager Bean。

Kafka Streams 的支持

Spring 专门针对 Apache Kafka 提供了工厂Bean去创建 StringBuilder 并管理 streams 的生命周期。 只要环境中引入依赖 kafka-stream 并且通过注解 @EnableKafkaStreams 启用 kafka stream功能， Spring Boot 的 auto-configures KafkaStreamsConfiguration 会自动配置对应的工厂环境。

如果要使用 Kafka Stream 你必须要提供 application id 和 bootstrap servers 的配置。 其中 spring.kafka.streams.application-id 如果没有设置会自动使用 spring.application.name 配置项代替。 spring.kafka.streams.application-id 可以全局配置使用一个，也可以针对特定的Streams 单独设置。 针对 Kafka Stream的设置都在 spring.kafka.streams.properties 命名空间之下。

测试 - 使用嵌入式Kafka

为了方便测试 Spring 提供了一个嵌入式的Kafka Broker 。要使用这个功能，首先需要包含 spring-kafka-test 依赖，同时 在测试类标记 @EmbeddedKafka.

使用时需要用嵌入式配置项来代替真实环境的 spring.kafka.bootstrap-servers 配置。

具体做法有如下集中方式：

  • 通过系统配置来覆盖
  • 使用注解 @EmbeddedKafka
  • 使用占位符在配置文件中配置

@EmbeddedKafka 注解使用说明

通常建议将规则用作@ClassRule，以避免在测试之间启动和停止 Kafka Broker 和 对于每个测试使用不同的主题。 从 Spring Boot 2.0 开始， 可以直接提供一个 EmbeddedKafkaBroker 的Bean来在多个测试之间复用 Broker。 为了方便使用，还可以使用 @EmbeddedKafka 来注册 EmbeddedKafkaBroker Bean.
