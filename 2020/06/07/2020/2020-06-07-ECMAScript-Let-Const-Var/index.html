const 声明

const声明一个只读的常量。一旦声明，常量的值就不能改变。const一旦声明变量，就必须立即初始化，不能留到以后赋值。 const命令声明的常量也是不提升，只能在声明的位置后面使用。

对于复合类型的变量，变量名不指向数据，而是指向数据所在的地址。const命令只是保证变量名指向的地址不变， 并不保证该地址的数据不变，所以将一个对象声明为常量必须非常小心
